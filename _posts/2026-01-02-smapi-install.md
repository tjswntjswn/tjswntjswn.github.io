맥북에서 스타듀밸리 모드를 즐기기 위해 스마피(SMAPI)를 설치하다 보면, 보안 문제로 인해 파일이 삭제되거나 실행되지 않는 오류가 자주 발생합니다.
저도 계속 실패하다가 **터미널 권한 설정**을 통해 해결했습니다. 저처럼 헤매지 않도록 설치 순서와 해결법을 정리해 둡니다.
<br>

---

## 1. SMAPI 다운로드

가장 먼저 넥서스모드에서 스마피 설치 파일을 다운로드합니다.

1. [SMAPI - Stardew Modding API](https://www.nexusmods.com/stardewvalley/mods/2400?tab=description) 사이트에 접속합니다.
2. **로그인**을 합니다. (계정이 없다면 회원가입 필수)
3. 상단 **FILES** 탭을 클릭합니다.
4. **Mod manager download** 버튼을 클릭합니다.
5. **Slow download** 버튼을 클릭하여 다운로드합니다.

---

## 2. 설치 파일 실행 (보안 우회)

맥의 보안(Gatekeeper) 때문에 그냥 실행하면 막힙니다. 아래 순서대로 진행하세요.

1. 다운로드한 파일의 **압축을 풉니다.**
2. SMAPI 4.3.2 installer 폴더로 들어갑니다.
3. install on macOS.command 파일을 **우클릭(또는 Control + 클릭) → [열기]**를 누릅니다.

> **참고:** 그냥 더블클릭하면 실행되지 않습니다.

### "확인된 개발자가..." 경고창이 뜨면서 실행이 안 된다면?
1. **[시스템 설정]** → **[개인정보 보호 및 보안]**으로 이동합니다.
2. 아래로 스크롤하여 **보안** 섹션에서 'install on macOS.command'의 사용이 차단되었습니다. 문구 옆의 **[확인 없이 열기]** 버튼을 클릭합니다.
3. 터미널 창이 뜨면 비밀번호를 입력하고 엔터를 눌러 설치를 진행합니다. (화면 안내에 따라 숫자 1 입력 후 엔터)

---

## 3. ★ 핵심 오류 해결 (터미널 권한 부여) ★

설치가 완료되었어도 게임 실행 시 **"파일이 손상되었습니다"** 또는 **"악성 코드가 포함되어 있습니다"**라며 실행이 안 되는 경우가 있습니다. 이 과정이 가장 중요합니다.

### 개발자 도구 권한 허용 (필수)
터미널 명령어를 입력하거나 스마피를 실행해도 해결되지 않는다면 이 설정이 꺼져있어서 그렇습니다.

1. **[시스템 설정]** → **[개인정보 보호 및 보안]**으로 들어갑니다.
2. **[개발자 도구]** 항목을 클릭합니다.
3. 목록에 있는 **터미널(Terminal.app)**의 스위치를 켜서 **사용 허용**을 해줍니다.
4. 터미널 명령어로 격리 해제
터미널(Terminal)을 열고 아래 명령어를 입력하여 MacOS 폴더 전체의 격리 속성을 제거합니다.
```
sudo xattr -cr "~/Library/Application Support/Steam/steamapps/common/Stardew Valley/Contents/MacOS"
```

---

## 4. 게임 실행 및 확인

1. 스팀(Steam)에서 **플레이** 버튼을 누릅니다.
2. 검은색 스마피 콘솔 창이 먼저 뜨고, 잠시 후 게임이 정상적으로 실행되는지 확인합니다.

<br>
> **요약:** 설치가 안 된다면 **시스템 설정에서 터미널 앱에 '개발자 도구' 권한을 줬는지** 꼭 확인하세요! 이것 때문에 몇 시간을 헤맸네요. 즐스듀 하세요! 🌾
